# CropCare-System

### Description
The backend service for the Crop Care project, built using Java and Spring Boot. It provides RESTful APIs for managing equipment, fields, and staff.

### Technologies Used
- Java
- Spring Boot
- Gradle
- Hibernate
- JPA
- Lombok
- Spring Security

### Setup Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/nethmiumaya/Crop-Care-BACKEND.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Crop-Care-BACKEND
   ```
3. Build the project:
   ```sh
   ./gradlew build
   ```
4. Run the application:
   ```sh
   ./gradlew bootRun
   ```

### API Endpoints
- **Equipment**
- **Monitory Log**
- **User**
- **Field**
- **Crop**
- **Vehicle**
- **Staff**
